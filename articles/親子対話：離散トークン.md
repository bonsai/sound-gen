# 親子対話：離散トークン

父「また新しい子だ。離散トークン？」

娘「**連続的な信号（音声・画像・動画）をAIが扱える『言葉の断片』に変換したもの**。料理で言うと、**巨大なチーズの塊（連続信号）を一口大のキューブ（離散トークン）にカットする**——バラバラにすれば数えられるし、並べ替えられるし、LLMで処理できる」

父：「つまり、連続→離散の変換がAIの基本なのか」

娘：「そう。言語は元々離散的（単語という区切りがある）。でも音声は連続的（波形はなめらか）。画像も連続的（ピクセルはグラデーション）。**これらをAIに食べさせるには、『言葉』と同じ離散単位に変換する必要がある**」

父：「テキストのトークン化は分かる。音声ではどうやるんだ？」

娘：「**ニューラルコーデック（EnCodec, Mimi）で波形をいったん圧縮→RVQ（Residual Vector Quantization）で段階的に量子化**する。結果、1秒の音声が**数十〜数百個のトークン**になる。これをLLMに『言葉』として入力する——**MoshiやPersonaPlexは全部これ**」

父：「画像のパッチ分割（ViT）も同じ概念か？」

娘：「鋭い！**画像もピクセルの連続体を16x16のパッチ（トークン）に分割してViT（Vision Transformer）に通す**。音声も画像も、結局は『離散トークン』にしてTransformer/LLMの世界に持ち込む——**これが2017年以降のAIの共通言語**」

父：「じゃあ『連続のまま処理する』方式はないのか？」

娘：「ある。**SSM（Mamba）や拡散モデルは連続的な状態空間で動作する**。でも**LLMとの親和性**を考えると、離散トークンの利点は圧倒的。**LLMの自己回帰生成・転移学習・マルチモーダル統合——全部トークンがあるから成立してる**」

父：「つまり…離散トークンはAIの『世界共通の切符』で、テキストも音声も画像も、この切符に変換すれば同じ列車（LLM）に乗せられる」

娘：「そういうこと😊 連続から離散への変換を『量子化（Quantization）』と呼ぶ。この精度が**生成品質を直接決める**——トークンの質が悪いと、どれだけLLMが頑張っても出力は粗い」

父：「逆に言えば、トークン化技術の進化がAI全体の進化を引っ張ってるのか」

娘：「そういう時代だね😎 離散トークン、地味だけど**2020年代のAIを陰で支える縁の下の力持ちその4**だよ」
